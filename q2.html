Tim Berners Lee

The World Wide Web

The World Wide Web (WWW) is an information system developed by Tim Berners Lee that is a means for content sharing and searching over the internet, which was developed in 1989. Originally, Tim Berners Lee was working on the web, with the goal to meet the demand for automated information-sharing between scientists in universities and institutes around the world.

HTML

HTML is the coding language invented by Tim Berners Lee, used to structure and create a web page. HTML is the code responsible for structuring content, paragraphs, a list of bulleted points, or adding images and data tables.

The Web Browser

A web browser is an application for accessing websites and the Internet. Tim Berners Lee was the original inventor of the web and how you navigated it with his web browser, called the WorldWideWeb, which he developed in 1990.
